# [StarStudy]

### 해당 서비스는 2024 경주 지역문제 해결 해커톤 [19.3]의 프로젝트 입니다.

---
## 서비스 소개(예시)
### [StarStudy]
- 수도권 비수도권의 학습권 격차를 줄이기 위해 학습공간 정보를 제보하여 격차를 줄이기위함 입니다.

---
## 기능(예시)
- 지도 기능을 통해 학습공간 위치 정보를 제공하여 학습 권유
- 학습공간 검색 기능을 통해 근처 학습공간의 주소을 텍스트로 확인
- 게시판을 이용하여 학습공간의 정보를 공유
